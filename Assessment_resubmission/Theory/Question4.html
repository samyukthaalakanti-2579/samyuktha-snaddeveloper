IMPLICIT CONVERSION In javascript conversion will happen without the developer noticing. conversion is the value type to the expected type under the hood. when we want to make a value type to special type this conversion will happen in javascript. Example: // Implicit conversion to string '25' + 15; // '2515' // Implicit conversions to a number 23 * '2'; // 46 23 - true; // 22 // true is converted into 1 true - null; // 1 false + undefined; // NaN // undefined into NaN // Implicit conversions in statements const arr = []; if(arr) { console.log('Hello World') }; // Hello World // Here the empty arr is converted into a Boolean true
